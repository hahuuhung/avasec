# Hướng dẫn Chạy và Triển khai SysAnti

Dự án đã được đóng gói thành công thành hai phần: **Server** và **Client**.

## 1. Yêu cầu hệ thống

- Hệ điều hành: Windows 10/11 (x64)
- Không cần cài đặt .NET Runtime (đã tích hợp sẵn).

## 2. Server (API & Web)

Thư mục: `Publish/Server`

### Cách chạy

1. Mở thư mục `Publish/Server`.
2. Chạy file **`SysAnti.API.exe`**.
3. Một cửa sổ console sẽ hiện ra, server sẽ lắng nghe tại `http://localhost:5000`.
4. Mở trình duyệt và truy cập: `http://localhost:5000`

### Tài khoản Admin mặc định

- Username: `admin`
- Password: `Minad@123`

## 3. Client (Ứng dụng Desktop)

Thư mục: `Publish/Client`

### Cách chạy

1. Mở thư mục `Publish/Client`.
2. Chạy file **`SysAnti.UI.exe`**.
3. Ứng dụng sẽ khởi động và kết nối tới Server (nếu Server đang chạy).

## 4. Dành cho lập trình viên (Developers)

### Source Code

Source code được lưu trữ trong Git. Để tiếp tục phát triển:

1. Clone repository hoặc mở thư mục `SysAnti`.
2. Mở `SysAnti.sln` bằng Visual Studio 2022.
3. Chạy Server: Set `SysAnti.API` làm Startup Project -> Run.
4. Chạy Client: Set `SysAnti.UI` làm Startup Project -> Run.

### Cấu trúc thư mục đóng gói (Publish)

```
Publish/
├── Server/           # Chứa SysAnti.API.exe và SysAnti.Web
│   ├── SysAnti.Web/  # Tài nguyên Web tĩnh
│   └── ...dll        # Các thư viện phụ thuộc
└── Client/           # Chứa SysAnti.UI.exe
```

### Git Note

Thư mục `Publish/` mặc định được ignore bởi git. Nếu bạn muốn lưu trữ bản build, hãy xóa dòng `Publish/` trong `.gitignore`.
