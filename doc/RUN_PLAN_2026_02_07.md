# Kế hoạch hướng dẫn chạy ứng dụng SysAnti
**Ngày giờ:** 2026-02-07 13:20
**Người thực hiện:** Antigravity AI

## 1. Mục tiêu
Cung cấp hướng dẫn chi tiết cách chạy ứng dụng Desktop (WPF) và Server Web (Node.js) cho dự án SysAnti.

## 2. Các bước thực hiện
### Bước 1: Chuẩn bị môi trường
- Kiểm tra .NET 9.0 SDK đã cài đặt chưa.
- Kiểm tra Node.js đã cài đặt chưa.
- Kiểm tra MySQL đã cài đặt chưa (dành cho Server).

### Bước 2: Chạy ứng dụng Desktop (WPF)
- Cách 1: Sử dụng lệnh `dotnet run --project SysAnti.UI`.
- Cách 2: Sử dụng file `run_admin.bat` (yêu cầu đã build ứng dụng).

### Bước 3: Chạy Server Web (Node.js)
- Di chuyển vào thư mục `SysAnti.Server`.
- Cài đặt dependencies bằng `npm install`.
- Cấu hình file `.env` nếu cần thiết (DB_USER, DB_PASSWORD).
- Chạy server bằng `npm run dev` hoặc `npm start`.

## 3. Thời gian dự kiến hoàn thành
- Tài liệu hướng dẫn: 10 phút.
- Hỗ trợ người dùng chạy thử (nếu cần): Phụ thuộc vào phản hồi.
