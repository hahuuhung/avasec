🎯 Hướng dẫn Test / Testing Guide
1. Đăng ký Tài khoản / Register Account
Nhập username (ví dụ: admin)
Nhập email (ví dụ: admin@test.com)
Nhập password (ví dụ: password123)
Nhấn "Register / Đăng ký"
Sẽ tự động được cấp giấy phép dùng thử 14 ngày!
2. Đăng nhập / Login
Nhập username và password vừa đăng ký
Nhấn "Login / Đăng nhập"
Vào màn hình chính!
3. Test Các Tính năng / Test Features
💾 Disk Cleanup:

Nhấn "Clean Now / Dọn ngay"
Xem kết quả MB đã dọn dẹp
⚡ RAM Optimization:

Nhấn "Optimize Now / Tối ưu ngay"
Xem MB RAM đã giải phóng
🔍 Virus Scanner:

Nhấn "Quick Scan / Quét nhanh"
Quét thư mục Downloads với 20+ virus signatures!
🔄 Virus Database Update:

Xem thông tin database ở header
Nhấn nút xanh "🔄 Update DB"
Kiểm tra phiên bản database tăng lên!
4. Kiểm tra Database / Check Database
Database được tạo tại:

%AppData%\SysAnti\avasec.db
Virus database được tạo tại:

%AppData%\SysAnti\VirusDatabase\virus_signatures.json
