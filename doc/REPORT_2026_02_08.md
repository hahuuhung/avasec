## 📝 Hướng dẫn sử dụng tính năng mới

### Gửi thông báo từ Admin
1. Truy cập `admin.html`.
2. Tại phần "Trung tâm Thông báo", nhập Tiêu đề và Nội dung.
3. Chọn đối tượng gửi: "Tất cả" hoặc nhập ID người dùng cụ thể.
4. Nhấn "Gửi".

### Backup Dữ liệu
Chạy lệnh sau trong thư mục `SysAnti.Server`:
```bash
node backup_db.js
```
File backup sẽ được lưu trong thư mục `backups/backup_YYYY-MM-DD...`.

### Kiểm tra Logs
Chạy lệnh:
```bash
node check_logs.js
```
