## 🎨 Design System Reference / Tham khảo Hệ thống Thiết kế

### Color Palette

- **Primary:** #4ecdc4 (Teal)
- **Secondary:** #ff6b6b (Red)
- **Warning:** #ffe66d (Yellow)
- **Success:** #95e1d3 (Green)
- **Info:** #aa96da (Purple)

### Typography

- **Font Family:** Segoe UI, Roboto
- **Heading 1:** 32px Bold
- **Heading 2:** 24px SemiBold
- **Heading 3:** 18px Medium
- **Body:** 14px Regular

### Spacing

- **Card Padding:** 16px
- **Section Margin:** 24px
- **Corner Radius:** 8px (standard), 12px (large)
