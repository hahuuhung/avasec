# Báo cáo Khắc phục Lỗi UI / UI Error Fix Report

(Created: 2026-02-06 11:52:00)

## 🐛 Vấn đề / Problem

Khi chạy `dotnet run --project SysAnti.UI\SysAnti.UI.csproj`, ứng dụng không hiển thị giao diện và crash với lỗi:

```
System.Exception: Cannot find resource named 'ProgressToWidthConverter'
```

## 🔍 Phân tích / Analysis

- **File gây lỗi:** `SysAnti.UI\Resources\CardStyles.xaml` (line 72)
- **Nguyên nhân:** Converter được khai báo trong `App.xaml` SAU khi merge resource dictionaries, nên khi XAML parser load `CardStyles.xaml` thì converter chưa tồn tại.

## ✅ Giải pháp / Solution

Thêm khai báo `ProgressToWidthConverter` trực tiếp vào đầu file `CardStyles.xaml`:

```xml
<ResourceDictionary xmlns:converters="clr-namespace:SysAnti.UI.Converters">
    <!-- Converters / Bộ chuyển đổi -->
    <converters:ProgressToWidthConverter x:Key="ProgressToWidthConverter"/>
</ResourceDictionary>
```
