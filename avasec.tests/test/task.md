Manual Test Checklist: SysAnti Ultimate
Use this checklist to verify the Main and Advanced functions of the redesigned SysAnti application.

1. 🏠 Main Interface (MainWindow)
 Startup: App launches directly to Dashboard (Guest Mode).
 User Icon: Located in top-right header. Click to open Login Dialog.
 Monitor: CPU and RAM bars update every second.
 Navigation: "Settings" (⚙️) icon opens Settings Window.
 Window Controls: Minimize (-) and Close (X) buttons work correctly.
2. 👤 Authentication Flow
 Login Dialog: Click User Icon -> Dialog appears.
 Validation: Try empty username/password -> Error message appears.
 Success: Enter valid details -> Dialog closes -> "Licensed to: ..." updates in Footer.
 Registration: Switch to Register tab -> Create account -> Receive success message.
3. ⚙️ Advanced Features (Feature Windows)
💾 Disk Cleanup
 Open: Click "Dọn dẹp ngay" button.
 UI: Dark Theme applied (Sidebar + Header).
 Scan: Click "Scan" (Quét) -> Progress bar runs -> Results listed.
 Clean: Select items -> Click "Clean" (Dọn dẹp) -> Success message.
⚡ RAM Booster
 Action: Click "Tối ưu ngay" button on Dashboard.
 Feedback: Message box shows freed MB (e.g., "Freed 150 MB").
 Update: Dashboard RAM bar drops after optimization.
🚀 Startup Manager
 Open: Click "Quản lý" (Startup Manager).
 UI: List of startup apps appears with dark styling.
 Toggle: Enable/Disable a startup item -> Status updates.
 Add: Try adding a new startup program (if implemented).
🛡️ Virus Scanner
 Quick Scan: Click "Quét nhanh".
 Progress: Status text updates ("Scanning...").
 Result: Message box shows scan summary (Files scanned/Threats found).
⚙️ Registry Tweaks
 Open: Click "Mở Tweaks".
 UI: Cards with "Enable/Disable" switches.
 Interaction: Toggle a tweak -> Success message.
🚀 Windows 11 Master
 Open: Click "Mở Công cụ / Open" on the Hero Card.
 UI: Gradient Header + Special "Gaming Mode" button.
 Back: Click "←" button -> Window closes.
4. 📝 Visual & UX
 Consistency: All windows share the same Dark Blue/Teal color palette.
 Fonts: Text is readable (Inter/Segoe UI), adequate contrast.
 Animations: Button hover effects work (color change/glow).
